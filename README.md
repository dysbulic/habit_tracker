# Running

* `sudo apt-get install couchdb`
* `sudo apt-get install python-setuptools`
* `sudo easy_install pip`
* `sudo apt-get install python-dev`
* `sudo pip install couchapp`
* `couchapp push . http://localhost:5984/habits`
